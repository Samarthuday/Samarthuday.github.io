Some Of My Work:

Event Management System

The Event Management System was a web-based application designed to help organizations plan, manage, and track events efficiently. It provided a comprehensive set of features to handle event registration, ticketing, scheduling, attendee management, and much more.

Music Recommender System

The project created a music recommendation system using Spotify's API and data analysis. It offered personalized song suggestions based on user preferences by analyzing factors like genre, tempo, popularity, danceability, and energy for accurate recommendations.

To-Do List

To-Do was a web app for task management, allowing users to add, update, delete tasks, and set reminders. It ran on XAMPP with MySQL for the database and Apache for the web server, and used JavaScript for reminders. User-specific tasks were securely stored in a hashed table upon login.

Real-Time Construction Dashboard

The Real-Time Construction Dashboard provided live updates and insights into construction projects. It tracked progress, managed tasks, and monitored key metrics to ensure efficient project management and timely decision-making.
